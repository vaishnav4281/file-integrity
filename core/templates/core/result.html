Understanding Entropy

Shannon Entropy measures the randomness of data within each block. A value of 0.0 indicates constant data, while 8.0 indicates maximum randomness.

01

Encryption Detection

High entropy (>7.5) often signals encrypted payloads or compressed data, common in ransomware attacks.

02

Structural Consistency

Stable entropy deltas across blocks confirm that the underlying data structure remains intact despite potential hash collisions.

Security Implications

Our heuristic engine analyzes mismatches to determine the likely intent of the modification.

03

Trojan Injection

Detected when file size increases while the header remains stable, suggesting appended malicious code.

04

Bit-Rot Analysis

Identified by single-bit flips with minimal entropy change, typically caused by hardware degradation.
